int read_conf ( dchat_conf_t *  cnf,
char *  filepath,
int *  required_set 
)

Reads a configuration file located at CONFIG_PATH.

Reads a dchat configuration file located at CONFIG_PATH and sets read values in the global config. All required commandline options are supported and can be specified in the config file as they would have been specified in the commandline. The only difference is that in the config a option is specified like this: "<option> <argument>\n".

Parameters
cnfPointer to global config
filepathPath to config file
required_setAmount of required options set and specified in the config file
Returns
0 on success, -1 if file could not be read (see: ERRNO of fopen) or a value > 0 indicating the line in the config file where an error occured
